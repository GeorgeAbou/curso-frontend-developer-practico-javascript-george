#Front Endp para Tienda en Línea con Menú Interactivo
Este proyecto es una tienda en línea simple con un menú interactivo, carrito de compras y detalles de productos, utilizando HTML, CSS y JavaScript.

Funcionalidades
Menú de Escritorio y Móvil: Navegación interactiva con menús adaptativos.
Carrito de Compras: Agregar y ver productos en el carrito.
Detalles de Producto: Ver detalles de cada producto al hacer clic.
Agregar Productos: Los productos se pueden agregar al carrito de compras.
Requisitos
Navegador web.
Instrucciones
Clona el repositorio:

bash
Copiar código
git clone https://github.com/tu-usuario/tienda-en-linea.git
Abre index.html en tu navegador.

Interacción:

Haz clic en productos para ver detalles.
Agrega productos al carrito desde las tarjetas de productos.
Usa el menú de navegación en el escritorio o móvil.
